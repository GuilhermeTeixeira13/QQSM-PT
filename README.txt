QQSM-PT
* This project is fully writen in portuguese * 

QQSM -> "Quem Quer Ser Milionário" 

Este projeto escrito em C tem como objetivo simular o jogo QQSM 

Como usar:
  - Após fazer download, colocar os ficheiros todos juntos no foler, entrar nesse folder (cd QQSM-PT) e executar o comando make (make).
